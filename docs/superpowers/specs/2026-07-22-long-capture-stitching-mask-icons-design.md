# 长截图拼接、输入蒙层与图标调整设计

## 背景

当前长截图已经能让滚轮事件到达选区下方的应用，但真实滚动后累计预览和最终结果仍只有第一帧。运行时诊断显示画面发生变化后，长截图状态进入 `warning`，而 `frameCount` 保持为 1、`stitchedHeight` 保持为选区高度。

根因是运动跟踪器仅在降采样图像的正负 32 行内搜索位移。真实鼠标滚轮一次可能移动超过该范围，新帧因此被判为不确定，没有进入拼接器。现有的四个一像素边框窗口也没有覆盖选区外区域，所以鼠标移出选区后会继续滚动其他底层内容。

## 目标

1. 支持真实鼠标滚轮产生的大步长滚动，并把所有成功匹配的稳定帧拼接成一张长图。
2. 实时累计预览、编辑结果、保存结果和完成复制必须来自同一份拼接数据。
3. 长截图期间，仅选区所在显示器显示 0.3 蒙层；选区是透明、可滚动的挖空区域，选区外阻止鼠标和滚轮穿透。
4. 长截图控制窗口始终位于蒙层上方并保持可交互。
5. 更换马赛克、长截图和翻译相关图标，并移除隐私工具入口。

## 非目标

- 不覆盖未包含选区的其他显示器。
- 不恢复自动滚动。
- 不通过全局滚轮钩子推算拼接位移。
- 不改变默认快捷键、普通截图编辑流程或扣子 OCR/翻译服务协议。

## 长图匹配与拼接

### 数据流

1. 进入长截图后采集第一帧，并把它作为拼接器的首块内容。
2. 用户在选区内手动滚动；观察器等待画面重新稳定。
3. 对新的稳定帧执行多尺度垂直匹配：
   - 先在完整有效高度内使用低分辨率灰度图进行粗搜索；
   - 再在最佳候选附近使用更高分辨率图进行精匹配；
   - 同时比较正向与反向结果，避免反向滚动被追加到长图。
4. 正向匹配成功后计算重叠高度，去除重复区域，并过滤可信的固定页头和页尾。
5. 新内容立即追加到分块拼接器，更新 `frameCount`、`stitchedHeight` 和预览 PNG。
6. 编辑、保存和完成操作调用同一个拼接器的最终输出；如果至少已有一帧，停止时保留当前完整或部分结果。

### 匹配原则

- 搜索范围由当前帧有效高度决定，不再限制为固定 32 行。
- 粗搜索负责覆盖大步长，精匹配负责避免重复纹理造成误判。
- 低纹理或低置信度帧不追加，并继续显示“请慢一点滚动”提示。
- 反向滚动暂停追加；重新滚回已接受尾部后才恢复正向拼接。
- 拼接高度必须以实际追加的像素行数为准，不能仅增加帧计数。

## 原生蒙层和输入隔离

### 布局

选区所在显示器被拆成四个原生蒙层窗口：

- 顶部：显示器顶边到选区顶边。
- 右侧：选区右边到显示器右边，垂直范围与选区相同。
- 底部：选区底边到显示器底边。
- 左侧：显示器左边到选区左边，垂直范围与选区相同。

四个窗口合起来覆盖显示器除选区外的全部区域，背景为 `rgba(0, 0, 0, 0.3)`。选区不创建覆盖窗口，因此底层应用能直接接收滚轮事件。

### 输入与层级

- 四个蒙层窗口不获取键盘焦点，但必须接收并吞掉鼠标和滚轮事件，不能设置鼠标穿透。
- 蒙层窗口排除在屏幕采集之外，不能污染长截图帧。
- 长截图预览和控制窗口在蒙层之后创建或显式提升到其上方，保证按钮可以点击。
- 绿色选区边框由蒙层窗口朝向选区的一侧绘制，不再需要独立的一像素穿透边框。
- Esc、取消、错误和正常完成都通过统一清理流程关闭四个蒙层及控制窗口，并恢复截图主窗口状态。

## 图标与工具栏

- 马赛克：`Grid2X2` 改为 Lucide `Blocks`。
- 长截图：`ImageDown` 改为 Lucide `GalleryVerticalEnd`。
- 工具栏文字识别：`ScanText` 改为 Lucide `Languages`，保留“文字识别”可访问名称。
- OCR 结果面板的“翻译为中文”按钮增加同一个 `Languages` 图标，使两处翻译语义一致。
- 删除工具栏“隐私工具”按钮、动作类型和点击入口；其余工具顺序自然前移。
- 图标继续统一使用 20px、1.8 描边和现有按钮尺寸。

## 错误处理

- 单帧暂时无法匹配时不终止会话，保留已拼接结果并显示慢滚提示。
- 连续无法匹配时不伪造高度或重复追加第一帧。
- 若最终只有第一帧，编辑、保存和完成仍返回有效单帧 PNG，但界面明确保留警告状态。
- 原生蒙层创建失败时，不隐藏主截图窗口，并关闭已经创建的临时窗口。

## 测试与验收

### 自动化测试

- 大于旧 32 行限制的正向位移可以被多尺度匹配识别。
- 可变大步长连续滚动可以产生多块拼接结果。
- 反向、重复纹理、低纹理和无关帧不会错误追加。
- 四个蒙层布局完整覆盖选区所在显示器且不与选区相交。
- 蒙层窗口接收输入，预览窗口保持可交互，全部临时窗口统一清理。
- 工具栏不再包含隐私工具，并使用新的三个图标语义。
- 预览输出和最终输出具有相同的累计尺寸。

### Windows 实机验收

1. 选区内连续滚动至少三次，确认底层目标应用滚动。
2. 确认 `frameCount > 1` 且 `stitchedHeight` 大于选区高度。
3. 确认右侧累计预览随接受帧增长。
4. 分别验证编辑、保存和完成得到的 PNG 高度与累计拼接高度一致。
5. 把鼠标移到选区外滚动，确认底层其他区域不滚动。
6. 确认选区所在显示器除选区外均为 0.3 蒙层，其他显示器不受影响。
7. 按 Esc 后确认所有蒙层和控制窗口关闭，再次截图为空选区。
